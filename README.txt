# Calculadora de Idade

Este programa foi desenvolvido utilizando PyQt5 para criar uma interface gráfica onde o usuário pode selecionar sua data de nascimento e visualizar sua idade atual.

## Funcionalidades

- Mostra a data atual na interface.
- Permite selecionar a data de nascimento através de um calendário.
- Calcula automaticamente a idade com base na data selecionada.
- Interface com cores personalizadas para uma melhor experiência visual.

## Como usar

1. **Selecionar a Data de Nascimento:**
   - Use o calendário à esquerda para escolher sua data de nascimento.

2. **Visualizar a Idade:**
   - A idade atual será exibida na área designada assim que a data de nascimento for selecionada.

3. **Data Atual:**
   - A data atual é mostrada no topo da interface.

## Requisitos

- Python 3.x
- PyQt5
